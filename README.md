# Crypto Exchange

A mock cryptocurrency app made with CoinGecko APIs.

* 📈 Graphs of prices of coins and OHLCV data.
*	🪙 Exchanging and Staking cryptocurrencies.
* 🖼️	Marketplace for NFTs.
* 💎 Sync wallets with MetaMask.
